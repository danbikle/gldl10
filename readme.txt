This is a simple demo of classifying flowers using a combo of iris.csv and graphlab deeplearning.

Installation:
  - Install Anaconda
  - Install graphlab:
    - https://dato.com/download/install-graphlab-create-command-line.html

Operation:
  - cd ~
  - git clone github.com/danbikle/gldl10
  - cd gldl10
  - ./gldl.bash

Refer to:
  - https://dato.com/products/create/docs/graphlab.toolkits.deeplearning.html
